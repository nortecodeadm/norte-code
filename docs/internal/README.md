# docs/internal — Documentação Estratégica e de Protocolo

Documentação estratégica e de protocolo do Norte Code. Estes arquivos descrevem **o projeto**, não o código. Para documentação técnica (arquitetura, interpretador, níveis, decisões, setup), ver `docs/` raiz.

## Conteúdo

- **NORTECODE_Briefing_MVP_v2_5.md** — Briefing de execução v2.5 do MVP (Claude Estrategista → Manus Executor, via Gui). Consolida estado real do projeto em Maio/2026, divisão de trabalho Gui × Executor, e o princípio de não-retroatividade entre níveis.
- **NORTECODE_Dev_Guide.md** — Guia de desenvolvimento local no Windows 11 do Gui: versões da stack instalada, ambiente Android (JDK, SDK, ANDROID_HOME), fluxo do dia a dia, builds.
- **NORTECODE_Pitch.md** — Pitch narrativo do produto: o que é o Norte Code, por que o nome, público (7-10 anos) e curva pedagógica (sequência → loops → condicionais → variáveis → funções).
- **NORTECODE_Protocolo_Colaboracao_IAs.md** — Protocolo v1.0 que define os papéis das três IAs (ChatGPT Analista, Claude Estrategista, Manus Executor) e como o Gui orquestra a equipe.
- **NORTECODE_Protocolo_Dev_Temporario.md** — Define como qualquer IA assume temporariamente o papel de executor no lugar do Manus, com Cenários A (acesso direto ao GitHub) e B (sem acesso direto).
- **NORTECODE_Roadmap_Narrativo.md** — Visão narrativa de longo prazo: as fases iniciais da jornada (Origem, Cuidado, Cidade), o sistema Mundo Permanente + Missões Especiais ecoando histórias bíblicas sem nomeá-las, e como a tese pedagógica se conecta com a evolução do mundo.
- **NORTECODE_Style_Guide_Visual.md** — Style guide visual v1.1: estilo flat-design contemporâneo com toques orgânicos, referências (Toca Boca, Sago Mini, Alto's Odyssey), paleta, e regras invioláveis pra geração de assets por IA.

## Quando consultar

Quando o briefing técnico de uma tarefa **não der contexto suficiente** sobre intenção de produto, tom, identidade visual, ou direção narrativa. O CLAUDE.md raiz cobre o protocolo operacional; estes arquivos cobrem a **substância** do projeto.
